# Security Policy

## Supported versions

`agenda4j` is pre-1.0. Security fixes are provided on the latest `0.1.x` version.

## Reporting a vulnerability

Please do not open a public issue for security vulnerabilities.

Send details to: `support@harutoo.com`.

Include:
- Affected version
- Reproduction details or PoC
- Impact assessment
- Suggested mitigation (if any)

You can expect an initial response within 5 business days.
